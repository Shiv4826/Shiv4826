What is Full-stack Developer?

A Full-Stack Developer is a programer or a developer who handeled the frontend and backend of website. A full stack developer also can develop both client and server software. In addition to mastering HTML and CSS, he/she also knows how to: Program a browser (like using JavaScript, jQuery, Angular, or Vue) Program a server (like using PHP, ASP, Python, or Node) for becoming the full-stack developer we should must know about the frontend development language including- HTML, CSS, JAVASCRIPT, REACT JS, ANGULAR JS. and also too know about the Backend development language including- NODE JS, PHP, PYTHON, JAVA, RUBY.
